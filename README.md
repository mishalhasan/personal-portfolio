This project is not just a portfolio — it is a demonstration of architectural thinking, design intentionality, and frontend system design.

---

## 🏗️ Architecture & Technical Direction

The site was originally built using vanilla CSS and later underwent a full architectural refactor to Tailwind CSS. The refactor focused on improving scalability, maintainability, and design consistency.

Rather than relying on template-based layouts, I engineered a custom structural system emphasizing:

- Reusable layout wrappers
- Fluid spacing and typography using `clamp()`
- Consistent design tokens
- Clear separation of structure and presentation

The result is a responsive, mathematically consistent layout that scales cleanly across devices.

### Core Technologies

- **Frontend:** HTML5, Tailwind CSS, JavaScript
- **Design Approach:** Wireframe-first development, fluid typography, custom spacing system
- **Tooling:** VS Code, Git, Tailwind JIT Compiler

## 🛠️ Local Development

### 1. Clone the repository

```bash
git clone https://github.com/mishalhasan/portfolio.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the Tailwind compiler

```bash
npx tailwindcss -i ./src/input.css -o ./styles.css --watch
```
